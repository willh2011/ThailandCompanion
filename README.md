# Thailand Companion

Thailand Companion is a platform that helps people explore, move to, and live in Thailand.

## Vision

Build the best relocation platform for Thailand.

## Features

- Interactive Maps
- Property Search
- AI Relocation Assistant
- Cost of Living
- Neighborhood Guides
- Visa Information
- Schools
- Hospitals
- Saved Places

## Tech Stack

Backend
- ASP.NET Core (.NET 10)

Frontend
- Next.js
- React
- TypeScript

Database
- PostgreSQL
- PostGIS

Maps
- Leaflet
