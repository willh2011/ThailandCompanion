# Thailand Companion

Thailand Companion is a platform that helps people explore, move to, and live in Thailand.

## Vision

Build the best platform for anyone interested in moving to Thailand.

## Planned Features

- 🏠 Property Search
- 🗺 Interactive Map
- 🤖 AI Relocation Assistant
- 💰 Cost of Living Calculator
- 🛂 Visa Information
- 🏥 Hospital Finder
- 🏫 School Finder
- 📹 Local Video Guides
- ⭐ Saved Places
